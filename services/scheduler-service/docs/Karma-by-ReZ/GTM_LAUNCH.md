# Karma by ReZ — GTM & Launch Plan

## 1. Pre-Build Checklist (Week 0)

Before writing a single line of code:

- [ ] 1 NGO committed (signed MoU, 3 events on calendar)
- [ ] 1 CSR partner with ₹2–5L pilot budget
- [ ] 1 redemption brand confirmed
- [ ] 3 events with dates, times, locations
- [ ] ReZ Wallet API tested for batch credit
- [ ] ReZ Auth SSO verified

> If any of the above is missing, do not start building.

## 2. NGO Sourcing Plan

### Target NGO Types (Fastest to Onboard)

- Environment: tree plantation, lake cleanup, waste collection
- Food: weekend food drives, community kitchens
- Education: weekend teaching, library support
- Health: medical camps, blood donation drives

### 1-Page MoU (Simplified)

```
NGO NAME agrees to:
1. Post impact events on ReZ Karma platform
2. Verify volunteer attendance via QR scan or attendance sheet
3. Approve volunteer participation within 48 hours of event
4. Allow ReZ to use event photos/metrics for reporting

ReZ agrees to:
1. Bring verified volunteers to your events at no cost
2. Provide digital impact reports with volunteer metrics
3. Convert volunteer hours into ReZ Coins (real monetary value)
4. No financial liability on NGO

Both parties agree:
- Either party can terminate with 30 days notice
- Data sharing follows applicable privacy laws
```

## 6. Success Metrics

| Metric | Target | How to Measure |
|--------|--------|---------------|
| Event participation rate | >60% of registered users show up | check-in / registered |
| Completion rate | >80% of check-ins get approved | approved / check-ins |
| Cost per volunteer | <₹50 | CSR spend / volunteers |
| Weekly active karma users | >50 by week 8 | DAU on karma tab |
| ReZ Coin redemption | >30% convert coins to spend | wallet transactions |
| NGO NPS | >7/10 | survey |

## 7. Bangalore Cold Start Strategy

### Use ReZ's Existing Merchant Network
- Many ReZ merchants are already aligned (organic food, eco-friendly, wellness)
- Recruit them as event hosts + redemption partners simultaneously

### College Campus Push
- Tie up with NIT Surathkal, IISc, Christ University, St. Joseph's College
- Students = highest engagement, lowest acquisition cost
- Student clubs = organic volunteer supply

### Anchor Event (Week 7 Launch Event)
- Partner with BBMP (Bruhat Bengaluru Mahanagara Palike)
- Large-scale lake cleanup or tree plantation
- Media coverage opportunity
- 200+ volunteers target
