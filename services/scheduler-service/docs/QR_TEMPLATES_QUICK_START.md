# QR Template Designer - Quick Start Guide

## 🚀 Getting Started (5 Minutes)

### Step 1: Access the QR Designer
1. Log in to restaurant dashboard
2. Go to **Settings** → **Menu QR Code**
3. Navigate to `/restaurant/qr-templates`

### Step 2: Choose Your Template
- **Classic:** Professional, minimalist (black & white)
- **Modern:** Contemporary, trendy (indigo gradient)
- **Food Theme:** Food-focused (warm orange tones)
- **Vibrant:** Bold, artistic (pink & magenta)
- **Custom:** Fully branded (your colors)

### Step 3: Customize Colors
- Click color picker or enter hex code
- Use 16 preset colors for quick selection
- Preview updates in real-time

### Step 4: Download
- Click **Download** button
- Choose SVG (vector, for print) or PNG (raster, for web)
- Save to device

### Step 5: Deploy
- **Print:** Add to menus, table cards, entrance
- **Digital:** Share on Instagram, Facebook, website
- **Email:** Send to customers, delivery partners

## 🎯 Pro Tips

### For Print
✓ Use SVG format (vector, scalable)
✓ Minimum size: 2 cm × 2 cm
✓ High contrast colors
✓ Test scanning before printing

### For Digital
✓ Use PNG format
✓ Place on solid background
✓ Larger = more scans
✓ Position near restaurant name

### Color Best Practices
**Good Contrast:**
- Black on white ✓
- Dark blue on light gray ✓

**Poor Contrast:**
- Light gray on white ✗
- Similar brightness levels ✗

## ❓ FAQ

**Q: Can I change templates later?**
A: Yes! Log in anytime, select new template, save.

**Q: What format for printing?**
A: SVG (vector) is best. PNG works but may pixelate.

**Q: Minimum QR code size?**
A: 2 cm × 2 cm minimum. Default 256px works.

**Q: Does it work on mobile?**
A: Yes! Fully responsive on all devices.

**Q: How do I make it scannable?**
A: Use high color contrast (dark on light).

## 🔧 Troubleshooting

**QR Not Generating?**
- Wait a moment
- Click Refresh button
- Clear browser cache

**Colors Not Saving?**
- Check internet connection
- Verify you're logged in
- Try again

**Download Button Disabled?**
- Wait for QR to generate
- Refresh page
- Try different browser

**QR Doesn't Scan?**
- Check color contrast
- Ensure size ≥ 2cm
- Test on different phone
